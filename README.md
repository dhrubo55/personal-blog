# Mohibul Hassan's Blog

Personal website and writing archive for Mohibul Hassan Chowdhury.

The site is built with [Hugo](https://gohugo.io/) and the [PaperMod](https://adityatelange.github.io/hugo-PaperMod/) theme, then deployed to Netlify.

## Development

Run a local preview:

```powershell
hugo server
```

Run a production build:

```powershell
hugo --gc --minify --enableGitInfo
```
