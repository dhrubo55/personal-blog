# Java Learning Atlas Page Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Redesign the existing 100DaysOfJava graph page into an atlas-style dashboard with a hero summary, journey timeline, concept paths, interactive graph, and focused post inspector.

**Architecture:** Keep Hugo as the static generator and keep Cytoscape as the primary graph renderer because the existing page already has graph data, filtering, selection, and layout behavior. Add an atlas page shell around the graph instead of replacing the graph engine; use Three.js only as a later isolated spike if a 3D graph clearly improves comprehension after the 2D atlas is working. Store all derived journey/concept metadata in `static/js/java-graph.js` so the Hugo JSON contract remains focused on posts and relationships.

**Tech Stack:** Hugo, PaperMod, Hugo templates, static JavaScript, Cytoscape 3.29.2, Lucide icons via CDN, CSS Grid/Flexbox, static JSON generated by Hugo.

---

## Brainstorm Decisions

- Direction: make the page feel like a learning atlas, not a generic graph demo.
- Primary screen: use a dense operational dashboard like the reference image: hero on top, navigation cards below, timeline left, concept paths and graph center, selected post inspector right.
- Renderer: keep Cytoscape for the real graph. Three.js is not needed for the first implementation because the graph is 2D relationship navigation, not a spatial 3D scene.
- Data: reuse the existing `nodes` and `edges` JSON. Derive years, concept paths, milestone cards, selected post details, and hero stats in JavaScript.
- Interaction: make graph selection drive the inspector, timeline, and concept path highlighting. Use URL hash selection like `#day-69` so the page can open directly to a post.
- Mobile: stack sections in reading order: hero, mode cards, selected post inspector, graph, timeline, concept paths.
- Visual style: dark background, restrained orange accent, thin graph lines, compact panels, no rounded oversized marketing cards.

  const renderConceptPaths = () => {

  const hydrateIcons = () => {
    if (!window.lucide) {
      return;
    }

    document.querySelectorAll("[data-lucide-button]").forEach((button) => {
      const iconName = button.getAttribute("data-lucide-button");
      if (!iconName || button.querySelector("svg")) {
        return;
      }
      const icon = document.createElement("i");
      icon.setAttribute("data-lucide", iconName);
      icon.setAttribute("aria-hidden", "true");
      button.prepend(icon);
    });

    document.querySelectorAll("[data-lucide-link]").forEach((link) => {
      const iconName = link.getAttribute("data-lucide-link");
      if (!iconName || link.querySelector("svg")) {
        return;
      }
      const icon = document.createElement("i");
      icon.setAttribute("data-lucide", iconName);
      icon.setAttribute("aria-hidden", "true");
      link.prepend(icon);
    });

    document.querySelectorAll("[data-lucide-icon]").forEach((button) => {
      const iconName = button.getAttribute("data-lucide-icon");
      button.innerHTML = `<i data-lucide="${escapeHtml(iconName)}" aria-hidden="true"></i>`;
    });

    document.querySelectorAll("[data-lucide-panel]").forEach((target) => {
      const iconName = target.getAttribute("data-lucide-panel");
      target.innerHTML = `<i data-lucide="${escapeHtml(iconName)}" aria-hidden="true"></i>`;
    });

    document.querySelectorAll("[data-lucide-inline]").forEach((target) => {
      const iconName = target.getAttribute("data-lucide-inline");
      target.innerHTML = `<i data-lucide="${escapeHtml(iconName)}" aria-hidden="true"></i>`;
    });

    document.querySelectorAll("[data-lucide-label]").forEach((target) => {
      const iconName = target.getAttribute("data-lucide-label");
      target.innerHTML = `<i data-lucide="${escapeHtml(iconName)}" aria-hidden="true"></i><span class="sr-only">${escapeHtml(target.textContent || iconName)}</span>`;
    });

    window.lucide.createIcons();
  };

```javascript
  const renderMiniMap = () => {
    if (!miniMapEl || !model) {
      return;
    }

    const featured = model.nodes
      .filter((node) => [1, 6, 24, 33, 52, 69, 75, 88, 95, 100].includes(Number(node.day)))
      .slice(0, 10);

    const width = 360;
    const height = 210;
    const center = { x: 150, y: 105 };
    const points = featured.map((node, index) => {
      const angle = (Math.PI * 2 * index) / Math.max(featured.length, 1);
      const radius = index % 3 === 0 ? 82 : 112;
      return {
        node,
        x: center.x + Math.cos(angle) * radius,
        y: center.y + Math.sin(angle) * radius
      };
    });

    const lines = points.map((point) => `<line x1="${center.x}" y1="${center.y}" x2="${point.x}" y2="${point.y}" />`).join("");
    const nodes = points.map((point) => `<circle cx="${point.x}" cy="${point.y}" r="${Number(point.node.day) === 69 ? 8 : 5}" />`).join("");

    miniMapEl.innerHTML = `
      <svg viewBox="0 0 ${width} ${height}" role="presentation">
        <g class="java-atlas-mini-map__lines">${lines}</g>
        <circle class="java-atlas-mini-map__core" cx="${center.x}" cy="${center.y}" r="31"></circle>
        <text x="${center.x}" y="${center.y + 6}" text-anchor="middle">Java</text>
        <g class="java-atlas-mini-map__nodes">${nodes}</g>
      </svg>
    `;
  };
```
